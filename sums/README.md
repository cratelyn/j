# ∑ sum examples

this directory contains programs that compute the sum of integers from 1 to 100 (inclusive).

###  running

**sh**

```sh
./sum.sh
```

**C**

```sh
gcc sum.c && ./a.out
```

**Rust**

```sh
rustc sum.rs && ./sum
```

```sh
rustc sum2.rs && ./sum2
```

```sh
rustc sum3.rs && ./sum3
```

**Julia**

```sh
julia sum.jl
```

**J**

```sh
jconsole < sum.j
```

**K**

```sh
k < sum.k
```
